Listas de Arquivos Glob e Glob Extendido


Description

Produzindo listas de arquivos com os recursos do glob do bash. O glob pode ser usado em qualquer comando que aceite arquivos! Importante: enquanto é possível listar diretórios ( com */ )ou todos os arquivos ( com * ) , não é possível listar some

Para tanto, você deve usar o 'test' do bash com a opção '-d' em um loop, da seguinte forma:

printf '%s\n' * | while read i; do [[ -d "$i" ]] && echo "$i"; done

Glob = Global, comando global
