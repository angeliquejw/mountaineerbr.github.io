Usando o Vanitygen Endereco Personalizado dos Bitcoins


Description

Vanitygen é um gerador de endereço de bitcoin personalizado de linha de comando. Se você está cansado dos endereços aleatórios e enigmáticos gerados pelos clientes regulares de bitcoin, pode usar o vanitygen para criar um endereço mais personalizado. Como alternativa, o vanitygen pode ser usado para gerar endereços aleatórios Ref: https://en.bitcoin.it/wiki/Vanitygen

Página do Github:
https://github.com/samr7/vanitygen

No Arch, Manjaro, etc, disponível no AUR.

Instalar no Debian e afins:
https://github.com/samr7/vanitygen/wiki/Installation

Binários pro Windows:
https://allprivatekeys.com/vanity-address

Fork mais atualizado (Vanitygen-plus):
https://github.com/exploitagency/vanitygen-plus

Para rodar, execute digitando o número 1 seguido de um palavra/padrão da sua escolha.

$ vanitygen 1Boat

Para gerar endereços aleatórios, coloque só o número 1.

$ vanitygen 1

Endereço com uma expressão específica:

$ vanitygen 1YTB

Endereço com padrão específico, insensível a maiúsculas/minúsculas:

$ vanitygen -i 1ytb

Para gerar endereço com uma expressão em qualquer posição, use a opção "-r":

$ vanitygen -r [bB]oat

OBS: "[bB]" é uma expressão regular do Perl que indica que naquela posição pode ser "b" ou "B", já que a opção "-i" do Vanitygen não pode ser usada com a opção "-r".

Lembrando que demora o quanto mais longo for o padrão a ser procurado. Também, algumas letras específicas podem demorar mais que outras.
